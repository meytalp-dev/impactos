# מישמיש · מבנה ה-Pack — "המגוון יושב בדאטה"
### ספק · 13.7.2026 · נגזר מ-HANDOFF + מטריצת הרכישה

> **העיקרון בשורה אחת:** התבנית (mechanic) = קוד. ה-pack = דאטה. **המטרה (target) היא שדה** — לא קוד.
> באבני יסוד `target = "מתחיל באות מ"` (פענוח). אצל מישמיש `target = "התמונה ששמעת את שמה"` (הבנה שמיעתית).
> אותה קונכייה בדיוק — שדה אחד השתנה.

---

## 1. שלוש שכבות המגוון (תזכורת ההכרעה)

| שכבה | כמה | היכן חי | עלות |
|---|---|---|---|
| מכניקות (structural) | ~6, תחום נעול | `js/templates/mechanic-*.js` | יקר → משקיעים ועוצרים |
| Packs חוויתי (experiential) | ∞ | `data/pack.*.json` | זול → כאן המגוון |
| שפת-הישרדות (רוחבי) | 1, משותף | `data/scaffold.json` | פעם אחת |

**החוק:** רעיון חדש → pack או פרמטר בתבנית, **לעולם לא קובץ חדש**. אם לא נכנס לאף תבנית → מכניקה חדשה-באמת, מוסיפים בהחלטה מכוונת.

## 2. סכמת ה-Pack (worked example — נושא 5, "השוק המשותף")

```jsonc
// data/pack.market.json
{
  "meta": {
    "id": "market",
    "topic": "אוכל וקניות",           // תפאורה בלבד
    "setting": "shared-market",        // → assets/backgrounds/shared-market.png
    "space": "shared",                 // "hara" | "shared"  ← מסגור-הגשר
    "amir": true,                      // דמות-העמית (פיגום עאמייה) פעילה
    "sound_wave": 3                    // גל פּ/בּ בשלב זה (1–4)
  },

  "lexicon": [
    {
      "id": "apricot", "he": "מִשְׁמֵשׁ", "gloss": "apricot",
      "img": "food/apricot.png", "emoji": "🍑",
      "audio_he": "lex/apricot-he",        // עברית = יעד
      "cognate_ar": "مشمش",                // ← שדה "בני-הדוד" (קוגנט)
      "audio_amiya": "lex/apricot-amiya",  // עאמייה = פיגום (אמיר)
      "gender": "m", "number": "sg",       // ← ציר דקדוק (grammar-toggle)
      "sound_tags": ["b_p_free"]           // אין פּ/בּ פה
    },
    { "id": "banana", "he": "בָּנָנָה", "sound_tags": ["b_initial"], "...": "" },
    { "id": "bread",  "he": "לֶחֶם",   "sound_tags": [],            "...": "" }
  ],

  "patterns": [
    {
      "id": "want", "frame": "אֲנִי רוֹצֶה ___",
      "intent": "לבקש", "structure": "אני רוצה",
      "slots": ["apricot","banana","bread"],
      "audio_frame": "pat/want",
      "recurs_in": ["food","play","class"]   // ← חוזרת ב-3 נושאים ≥ (עמוד השדרה)
    },
    { "id": "howmuch", "frame": "כַּמָּה זֶה עוֹלֶה?", "intent": "לשאול", "...": "" }
  ],

  "contrasts": [                              // ← ציר הצליל
    { "pair": ["בַּ","פַּ"], "words": [["בָּנָנָה","פָּנָס"]], "wave": 3 }
  ],

  // ── הלב: אותו lexicon מזין 6 "שאלות" שונות ──
  "target": {
    "swim-hunt":      "audioWord → lexeme.img",       // שמעת → מצא תמונה
    "same-different": "contrast.pair discrimination",  // אותו/שונה
    "missing-slot":   "pattern.frame → choose slot",   // מה חסר בתבנית
    "grammar-toggle": "lexeme.gender/number match",    // מין/מספר/הוא-היא
    "follow-instr":   "lexeme + preposition",          // איפה / סדר
    "roleplay":       "pattern.frame produced to Amir" // turn עם אמיר
  }
}
```

**מה כל תבנית קוראת מה-pack (השחור-על-לבן):**

| תבנית | קוראת מה-pack | כותבת ל-BKT | רמה |
|---|---|---|---|
| swim-hunt | `lexicon` + `target.swim-hunt` | זיהוי-בשמיעה פר-lexeme | רצפטיבי |
| same-different | `contrasts` | הבחנת-צליל פר-pair | רצפטיבי |
| missing-slot | `patterns` | תבנית פר-pattern.id | מפיק-בתמיכה |
| grammar-toggle | `lexicon.gender/number` | מפת-מרחקים פר-מימד | מודרך |
| follow-instr | `lexicon` + prepositions | ביצוע-הוראות | קליטה |
| roleplay | `patterns` + `amir` | הפקה-באינטראקציה | אקטיבי |

התפאורה (`setting`, `emoji`, אודיו) מתחלפת פר-נושא. **8 packs × 6 תבניות = 48 חוויות, 6 קבצי-קוד.**

## 3. שני משחקוני-דגל — השראה מפנס + מערה

### 🔦 "מִשְׁמֵשׁ וְהַפַּנָּס" (מהפנס) — גילוי → המיפוי
מישמיש מטייל בחצרות בשעת ערב עם פנס; הילד **גורר את קרן האור באצבע** ומגלה חפצים בחושך; אוסף לתוך הסל את מה ש**שמע את שמו**.

- **מ-`flashlight-demo` עובר as-is:** מסכת-הקרן (`--mask radial`), גרירת-אצבע, ה-reveal, איסוף-לסל (net→basket), ה-`startle` שבו פריט-מסיח בורח בנגיעה שגויה (בלי אדום/ירוק), hint-pulse אחרי 9ש', תגובות-דמות (`noni-*` → `mishmish-*`), name-pop.
- **מה מוחלף (שדה אחד):** `target: "מתחיל באות מ"` → `target: "audioWord → lexeme.img"`. פלוס: לחישת-קוגנט (אמיר: مشمش → מישמיש), ומסיחי-פּ/בּ (בָּנָנָה מול פָּנָס שניהם נחשפים → בוחרים את מה ששמעו).
- **למה מושלם למיפוי:** החושך מונע לראות את כל האפשרויות בבת-אחת → מכריח הקשבה + נותן סוכנוּת-גילוי. probe שקט (מה נמצא מהר, מה דרש רמז) = אות אדפטיבי בלי תחושת-מבחן.

### 🕳️ "הַמַּעֲבָר לַשְּׁכוּנָה הַמְּשׁוּתֶּפֶת" (מהמערה) — בוס מצטבר
מישמיש חוצה מ"החצר שלי / حارتي" אל המרחב המשותף דרך שערים; כל שער = ציר אחד; **שומר-שער ידידותי (אמיר) פותח רק כשמשתמשים נכון** בתבנית/צליל.

- **מ-`cave-codex` עובר as-is:** ה-journey-track שמתמלא בתחתית, דמות-השומר + תגובותיה (`react/happy/startled/awake`), גרירת-חבל-אל-תמונה, איסוף מצטבר, ה-reveals הקולנועיים (light-shaft, gem-rise, shockwave), swim→walk transitions בין שערים.
- **מה מוחלף:** target פר-חדר = ציר (שער-צליל → שער-תבנית → שער-אינטראקציה) במקום "אבן-אות"; `guardian-message` = בקשה מדוברת שהילד ממלא ("תְּנִי לִי מִשְׁמֵשׁ").
- **למה מושלם כבוס-נושא:** מבנה-בוס + דמות-לרַצות + מפה-שמתמלאת + שיא-קולנועי — **והמסגור-הנרטיבי (מעבר בין שני מרחבי-הילד) עצמו הוא מבנה-המשחק.** התפאורה עושה עבודה כפולה.
