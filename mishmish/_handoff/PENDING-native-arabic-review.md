# 🟢 עאמייה ממתינה לבודק/ת ילידי/ת — דגל פתוח

**מצב (13.7):** אין כרגע בודק/ת ילידי/ת. החלטת מיטל: להתקדם — כל טקסט עאמייה **נשאר מגודר** (`ar_verified:false`, לא מוצג ולא מושמע), נשמר כטיוטה, **ומסומן כאן לבדיקה עתידית**. אין להעביר ל-`true` בלי אוזן ילידית (דיגלוסיה: עאמייה ≠ פוצחא — חובה בודק/ת מהמגזר).

**איך לבדוק כשיש בודק/ת:** דרך `tools/arabic-review-tool.html`. אחרי אישור פר-מחרוזת → להעביר את ה-flag ל-`true` במקור המתאים, ולהריץ `node src/js/_build-data.js`.
