## 4 שאלות מאושרות וקיימות בהקלטה

```
🟡 הסתגלות:   "האם נוני מרגיש טוב כשהוא בא לשונית בבוקר?"
🌸 חברות:      "האם יש לנוני עם מי לשחק בשונית?"
💜 שפה רגשית: "כשנוני כועס, האם יש משהו שעוזר לו להירגע?"
💚 שמחה ללמוד: "האם כיף לנוני לגלות משהו חדש?"
```

framing מרכזי = **"עזרי לנוני להבין את השונית"** — הילד.ה מדבר על נוני, לא על עצמה. זה ה-anti-bias המוטמע (other-perspective במקום self-rating).
