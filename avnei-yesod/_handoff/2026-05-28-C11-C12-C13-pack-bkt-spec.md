## §1 — הקשר: איפה C.11/12/13 יושב

| שאלה | תשובה |
|---|---|
| מה זה Pack? | יחידת תוכן חודשית. 11 פאקים בשנת תשפ"ז (ספט-יוני, ללא יולי-אוג). |
| מי כותב את התוכן של ה-Pack? | מיטל / צוות פדגוגי. JSON ידני. |
| מה זה Tier? | רמת קושי בתוך ה-Pack. **4 Tiers** (1=Basic, 2=Core, 3=Advanced, 4=Master+Mix). |
| מי בוחר את ה-Tier לילדה? | **המערכת אוטומטית** (לפי BKT), **המורה רואה** ב-teacher-rama **ויכולה להחליף ידנית**. |
| איזה BKT הוא הקובע? | **תלוי ב-Pack** — letters-focused (sub-BKT פר אות) או strand-focused (BKT-per-strand). |

**עקרון מנחה:** ילדה נכנסת ל-pack חודשי → מקבלת **Tier מתאים לה** (לא לכיתה) → ילדה אחרת באותו pack יכולה לקבל Tier אחר.

### 3.2 — שדות חובה

- `pack_id` (string, unique)
- `month_index` (1-12)
- `focus_mode` ("letters" או "strand")
- `primary_strand` (1-5)
- `letters_in_focus` (אם letters) **או** `strand_breakdown` (אם strand)
- `tiers` (object עם 4 keys: "1","2","3","4")
- כל tier: `items` (array)

### 3.3 — שדה strand_breakdown (אם focus_mode=strand)

```json
"strand_breakdown": {
  "skills": ["מודעות פותח/סוגר", "הברות", "פונמות"],
  "weight_per_skill": { "מודעות פותח/סוגר": 0.4, "הברות": 0.4, "פונמות": 0.2 }
}
```

זה מקרה ינואר-יוני. ה-bridge יידע לבחור Tier לפי BKT-per-strand של הסטרנד הראשי, לא לפי letters.

## §4 — C.12: Bridge (`pack-bkt-bridge.js`)

מיקום: `avnei-yesod/underwater-app/js/shared/pack-bkt-bridge.js`

### 4.1 — API ראשי

```js
window.AvneiPackBridge = {
  loadPack(packId)                          // → pack object (מ-/curriculum/packs/)
  loadCurrentPack(date)                     // → pack של החודש הנוכחי
  selectTierForStudent(studentId, packId)   // → { tier: 1-4, reason, confidence }
  getItemsForStudent(studentId, packId)     // → array of items מהטיר המתאים
  overrideTier(studentId, packId, tier)     // למורה — שמור ידנית
  clearOverride(studentId, packId)
  TIER_THRESHOLDS                           // const exported
};
```

### 4.2 — `selectTierForStudent` (הלב)

```js
function selectTierForStudent(studentId, packId) {
  const pack = loadPack(packId);

  // 1. בדיקת override ידני של מורה
  const override = getOverride(studentId, packId);
  if (override) {
    return {
      tier: override.tier,
      reason: `מורה הגדירה ידנית ב-${formatDate(override.date)}`,
      source: 'manual',
      confidence: 'high'
    };
  }

  // 2. חישוב אוטומטי לפי focus_mode
  let pKnown, confidence;

  if (pack.focus_mode === 'letters') {
    // משוקלל פר 4 אותיות (או n אותיות) הפאק
    const letterDist = AvneiBKT.getLetterMasteryDistribution(studentId);
    const focusLetters = pack.letters_in_focus.map(l => letterDist[l]);
    const valid = focusLetters.filter(l => l && l.attempts >= 5);

    if (valid.length === 0) {
      return { tier: 1, reason: 'אין דאטה — מתחילים מ-Tier 1', source: 'cold-start', confidence: 'low' };
    }
    pKnown = avg(valid.map(l => l.pKnown));
    confidence = valid.length === pack.letters_in_focus.length ? 'high' : 'med';

  } else { // strand
    const strands = AvneiBKT.getStudentStrands(studentId);
    const strand = strands[pack.primary_strand - 1];
    if (!strand || strand.attempts < 10) {
      return { tier: 1, reason: 'אין דאטה — מתחילים מ-Tier 1', source: 'cold-start', confidence: 'low' };
    }
    pKnown = strand.pKnown;
    confidence = strand.attempts >= 30 ? 'high' : 'med';
  }

  // 3. בחירת Tier לפי thresholds
  const tier = pickTier(pKnown);
  return {
    tier,
    reason: `${pack.focus_mode === 'letters' ? 'ממוצע על ' + pack.letters_in_focus.join(',') : 'סטרנד ' + STRAND_NAMES[pack.primary_strand]}: p=${pKnown.toFixed(2)}`,
    source: 'auto',
    confidence,
    pKnown
  };
}
```

### 4.3 — `TIER_THRESHOLDS` (ניתנים לכיול בפיילוט)

```js
const TIER_THRESHOLDS = {
  T1_MAX: 0.30,   // p<0.30 → Tier 1 (פערים)
  T2_MAX: 0.60,   // 0.30-0.60 → Tier 2 (חלקית)
  T3_MAX: 0.85,   // 0.60-0.85 → Tier 3 (טובה)
  // ≥0.85 → Tier 4 (מצוינת)
};

function pickTier(p) {
  if (p < TIER_THRESHOLDS.T1_MAX) return 1;
  if (p < TIER_THRESHOLDS.T2_MAX) return 2;
  if (p < TIER_THRESHOLDS.T3_MAX) return 3;
  return 4;
}
```

**הערה לפיילוט:** הספים האלה הם הצעה ראשונית. צריך לכייל לפי תוצאות פיילוט אמיתי. סוכן 8 לא ישנה אותם — רק חושף קבועים `TIER_THRESHOLDS` ב-API.

### 4.4 — `getItemsForStudent`

```js
function getItemsForStudent(studentId, packId) {
  const { tier } = selectTierForStudent(studentId, packId);
  const pack = loadPack(packId);
  return pack.tiers[tier].items;
}
```

### 4.5 — Override ב-localStorage

```js
const OVERRIDE_KEY = 'pack-overrides';
// schema: { [studentId]: { [packId]: { tier, date, author } } }

function overrideTier(studentId, packId, tier) {
  const all = JSON.parse(localStorage.getItem(OVERRIDE_KEY) || '{}');
  if (!all[studentId]) all[studentId] = {};
  all[studentId][packId] = { tier, date: Date.now(), author: 'teacher' };
  localStorage.setItem(OVERRIDE_KEY, JSON.stringify(all));
}
```

## §5 — C.13: Item Tagging

### 5.1 — Manual tagging (החלטה ראשית)

כל פריט ב-pack JSON מסומן ידנית עם `tier` ע"י כותב התוכן (מיטל/צוות). אין auto-classification בסקופ C.13.

### 5.2 — Schema לפריט

```json
{
  "item_id": "pack9-t2-001",          // unique
  "tier": 2,                            // 1-4
  "letter": "ש",                        // אם letters-focused
  "skill": "מודעות פותח/סוגר",        // אם strand-focused
  "mechanic": "tap-all",                // tap-all | pick | memory-pair | sort-by-letter
  "type": "new",                        // "new" | "review"
  "source_letter": "מ",                 // רק אם type=review (איזה אי קודם)
  "challenge": "with-niqud",            // אופציונלי
  "rama_task_alignment": 1,             // קיים מ-A0.2
  "peima_target": 1                     // קיים מ-A0.2
}
```

### 5.3 — Validation

סוכן 8 יבנה `scripts/validate-pack.js`:

```js
node validate-pack.js september-2026.json
```

בדיקות:
- כל item יש tier 1-4
- אם focus_mode=letters → letters_in_focus לא ריק, כל item.letter ב-letters_in_focus או type=review
- Tier 4 — `items_distribution.new + items_distribution.review === 1.0`
- כל mechanic ב-{tap-all, pick, memory-pair, sort-by-letter}
