## 3. סדר הבנייה המומלץ

### שלב 1 — להשלים קבוצת בועות (3 אותיות חסרות)
**אותיות:** ל · נ · א
**עבודה לאות:** העתקה של shin.json + הרצת `generate-<letter-key>-demo-audio.py` (שכפול של `generate-shin-demo-audio.py` עם החלפת LETTER_KEY והשורות).

זה הכי קל ויעיל למקצה הראשון — אותה תשתית, אותו ויזואל, רק 3 אותיות חדשות.

**אומדן:** ~1.5-2 שעות (3 אותיות × 30-40 דק'). אישור: שיחקה אחת מהן מקצה לקצה.

### שלב 2 — לבנות תשתית theme-aware
**מטרה:** game-shell.js יודע לקרוא `theme` מה-JSON ולהעביר ל-mechanic. mechanic-tap-all מוסיף `theme-<name>` class על `.tap-field`. CSS מטפל בכל ארבעת ה-themes.

**קבצים שמשתנים:**
- `js/templates/game-shell.js` — קריאה של `cfg.theme` והעברה ל-handler.mount
- `js/templates/mechanic-tap-all.js` — הוספת `theme-<theme>` class
- `css/game-shell.css` — 4 theme blocks (`.theme-bubbles` כבר קיים בפועל, רק לסמן; להוסיף `.theme-stars`/`.theme-shells`/`.theme-fish` עם SVG inline בתור CSS `background-image: url("data:image/svg+xml;...")`)

**הצורה SVG-inline (לא PNG!):**
- כוכב: 5 קודקודים, גרדיאנט זהוב-כתום, גודל 70% מהמסגרת, האות בפנים בכחול-ים כהה
- צדף: צורת מניפה (scallop) עם חריצים, פסטל ורוד/אפרסק, האות בפנים
- דג: צללית דג פשוטה (גוף אובלי + זנב משולש), צבע תכלת-טורקיז, האות במרכז הגוף

**אומדן:** ~3 שעות (תכנון + מימוש 3 SVG + ניסיון על אות-בודקה).

### שלב 3 — לבנות 4 אותיות בקבוצת כוכבים (Pilot ויזואל)
**אותיות:** ז · י · ו · ה
**עבודה:**
1. כתיבת `generate-stars-audio.py` (שכפול של generate-shin-demo, עם 2 משפטים כלליים + 4 משפטי find)
2. הרצה → 6 קבצי MP3 חדשים (1 quest + 1 mission + 4 find)
3. העתקה של 4 קבצי JSON עם `theme: "stars"`, החלפת letter/letter_key/distractors/quest_id
4. **בדיקה ידנית מקצה לקצה** של אחת מהאותיות מול מיטל לאישור

**אומדן:** ~4 שעות. ⚠️ **לעצור ולקבל אישור ממיטל לפני הקבוצות הנוספות.**

### שלב 4 — צדפים (4 אותיות)
**אותיות:** ס · ע · צ · ט
שכפול של שלב 3 על נושא "צדפים". ~3.5 שעות.

### שלב 5 — דגים (5 אותיות, צורה מורכבת)
**אותיות:** ד · ג · ח · פ · כ
שכפול. ~4 שעות (אות נוספת + צורה מורכבת יותר).

## 4. שורות האודיו לכתיבה ב-Python (תבנית פר נושא)

### כוכבי הים
```python
LINES = {
    "intro-stars-quest":   "נוֹנִי גִּלָּה כּוֹכָבִים זוֹהֲרִים בַּיָּם, וְלִכֹל כּוֹכָב יֵשׁ אוֹת!",
    "intro-stars-mission": "בּוֹאוּ תַּקִּישׁוּ עַל כֹּל הַכּוֹכָבִים עִם הָאוֹת <LETTER>, "
                           "וְכֹל אֶחָד שֶׁתַּדְלִיקוּ יְאִיר עוֹד פִּנָּה.",
    # פר אות:
    "find-zayin": "מִצְאוּ אֶת כֹּל הַכּוֹכָבִים עִם הָאוֹת זַיִן בַּיָּם.",
    "find-yud":   "מִצְאוּ אֶת כֹּל הַכּוֹכָבִים עִם הָאוֹת יוּד בַּיָּם.",
    "find-vav":   "מִצְאוּ אֶת כֹּל הַכּוֹכָבִים עִם הָאוֹת וָו בַּיָּם.",
    "find-hey":   "מִצְאוּ אֶת כֹּל הַכּוֹכָבִים עִם הָאוֹת הֵא בַּיָּם.",
}
```

⚠️ **`intro-stars-mission` מנוסח עם `<LETTER>` placeholder** — בפועל יש משפט מתקבע אחד לקבוצה (מציג את **שם הנושא**, לא האות), אז כל 4 האותיות חולקות אותו. שגיאתי הראשונית בטיוטה — לתקן.

### תיקון: `intro-<theme>-mission` לא מזכיר אות
```python
"intro-stars-mission": "בּוֹאוּ תַּקִּישׁוּ עַל הַכּוֹכָבִים בַּיָּם, "
                       "וְכֹל אֶחָד שֶׁתַּדְלִיקוּ יְאִיר עוֹד פִּנָּה.",
```

הפרטים על איזו אות לחפש מגיעים מ-`find-<letter-key>.mp3` בתוך המשחק עצמו (תוך 300ms אחרי start).

**`find-<letter-key>` תבנית:** `"מִצְאוּ אֶת כֹּל ה-<THEME_NOUN> עִם הָאוֹת <LETTER_NAME> בַּיָּם."`
דוגמה: `מִצְאוּ אֶת כֹּל הַצְּדָפִים עִם הָאוֹת סָמֶ״ךְ בַּיָּם.`

⚠️ **כללי TTS שאסור לשכוח (מ-[[feedback-tts-hebrew-niqud-pitfalls]]):**
1. כותבים **כֹּל** בחולם (לא כָּל)
2. AvriNeural בלבד · rate -10%
3. הניסוח "הצדפים עם האות X" — לא "צדפי X" (סמיכות אקדמית)
4. שם האות **בכתיב מלא** ("סָמֶ״ךְ", "ה ֵא", "ק ֹו ף") — לא "ק"
