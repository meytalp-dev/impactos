# אבני יסוד — מערכת דיפרנציאלית לכיתות א'-ב'

> **שם מוצר זמני.** מערכת שמלווה את המורה בכיתה הטרוגנית בהקניית קריאה וכתיבה.

---

## מה זה

מערכת **דיפרנציאלית למורה** של כיתות א'-ב', שנותנת לה מענה ליצירת מסלולי למידה אישיים בכיתה הטרוגנית — תוך התבססות על:

1. **תוכנית הלימודים של משרד החינוך** (תשפ"ו)
2. **אבני יסוד** — מאגר ידע פדגוגי שנאסף ממקורות סמכותיים (מטח/קסם וחברים, אופק, נגבה, השוואה לעולם)
3. **22 קטגוריות דקדוק-תחביר** של כיתה א'

- **לא** לקשיי קשב וריכוז
- **לא** לקות למידה

זה מוצר ל**מורה בכיתה רגילה הטרוגנית**, ולתלמידים ברצף תקין.

## כללי-יסוד (חובה)

### 1. אימות אוצר מילים
כל מילה שמופיעה כתשובה נכונה — חייבת להיות ב-[`curriculum/vocab-bank.json`](./curriculum/vocab-bank.json). לא להמציא.

### 2. הירארכיית מקורות סמכותיים
משרד החינוך → מטח (קסם וחברים) → אופק / נגבה → ספרות מחקרית.
בכל קביעה פדגוגית — לתייג מקור.

### 3. פרטיות תלמידים
תלמיד רואה רק את עצמו. רק המורה רואה את הכיתה.

### 4. הוספת ידע ל-KB
טריגר: "תוסיף לאבני יסוד" / "תוסיף לידע הפדגוגי" →
1. סרוק מקור
2. בנה JSON ב-`curriculum/knowledge-base/sources/`
3. עדכן `curriculum/knowledge-base/index.html`
4. עדכן `mandate` JSON רלוונטי ב-`synthesis/` אם נדרש
5. commit + push

### 5. שמות באנגלית
שמות קבצים ותיקיות — slug-באנגלית (`avnei-yesod`, לא `אבני-יסוד`). תוכן בעברית.
