> 🚨 **מסמך-אם:** [architecture-mvp.md](./architecture-mvp.md) — ארכיטקטורת ה-MVP המעודכנת (21.5.2026). היכן שיש סתירה ל-spec.html או למסמכים אחרים — architecture-mvp.md גובר.

## מה זה

מערכת **תרגול דיפרנציאלית** במסך, של 15-25 דקות ביום בתוך השיעור, שמלווה את המורה בכיתה הטרוגנית בהקניית קריאה. **לא** תוכנית-הוראה, **לא** ספר לימוד דיגיטלי, **לא** מערכת ניהול שיעורים — מערכת תרגול בלבד.

מבוססת על:

1. **תוכנית הלימודים של משרד החינוך** (תשפ"ו, פרופ' מיכל שני) — *משה"ח קובע, ראמ"ה בודקת*
2. **5 סטרנדים BKT** (פונולוגיה · מורפולוגיה · שפה דבורה · קריאה+הבנה · כתיבה) — מודל סטטיסטי דיפרנציאלי. ראה [architecture-mvp.md](./architecture-mvp.md). תיעוד פדגוגי של 9 המיומנויות של מטח: [literacy-grade1-2-yearly.md](./curriculum/literacy-grade1-2-yearly.md)
3. **22 איים — instructional milestones** (מטח · Share & Bar-On Triplex · Ravid · Kim IDL · ראמ"ה 2025)
4. **מנוע אדפטיבי:** BKT-per-strand + sub-BKT פר 22 אותיות + EPA תיאורי + mastery משולש (BKT + שטף + רף ראמ"ה)
5. **אבחון רב-ממדי** — 8 ממדים פדגוגיים + 5 ממדי SEL (בעתיד)

הליבה הפדגוגית במסמך `curriculum/pedagogy-integration-framework.md`. הליבה הטכנית במסמך `architecture-mvp.md` + `underwater-app/js/shared/`.
טעויות LLM אסור-לחזור-עליהן (Shatil/Bar-On/18 אותיות): `curriculum/llm-pitfalls.md`.

## כללי-יסוד (חובה)

### 1. אימות אוצר מילים
כל מילה שמופיעה כתשובה נכונה — חייבת להיות ב-[`curriculum/vocab-bank.json`](./curriculum/vocab-bank.json). לא להמציא.

### 2. הירארכיית מקורות סמכותיים
משרד החינוך → מטח (קסם וחברים) → אופק / נגבה → ספרות מחקרית.
בכל קביעה פדגוגית — לתייג מקור.

### 3. פרטיות תלמידים
תלמיד רואה רק את עצמו. רק המורה רואה את הכיתה.

### 4. הוספת ידע ל-KB
טריגר: "תוסיף לאבני יסוד" / "תוסיף לידע הפדגוגי" →
1. סרוק מקור
2. בנה JSON ב-`curriculum/knowledge-base/sources/`
3. עדכן `curriculum/knowledge-base/index.html`
4. עדכן `mandate` JSON רלוונטי ב-`synthesis/` אם נדרש
5. commit + push

### 5. שמות באנגלית
שמות קבצים ותיקיות — slug-באנגלית (`avnei-yesod`, לא `אבני-יסוד`). תוכן בעברית.
