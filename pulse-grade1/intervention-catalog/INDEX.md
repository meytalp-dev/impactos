**חבילת התערבות אוטומטית (Tier 1 universal):**

### 🍯 חזרה מחופש (אחרי חנוכה, פסח, סוכות, קיץ)

**מצב:** ירידה אוטומטית של 0.3-0.5 SD בממדים A + D ב-2 שבועות אחרי חזרה. זה רגיל ולא צריך פאניקה.

**חבילת התערבות מתוזמנת (מופעלת אוטומטית):**

2. **שאלת השבוע**: "מה הדבר הכי טוב שעשית בחופש?" ([cat-06](cat-06-curiosity-interest.md#מתכון-1-שאלת-השבוע))
3. **תיק הצלחות** ([cat-07](cat-07-capability-success.md#מתכון-1-תיק-הצלחות)) — להזכיר מה השגנו לפני החופש
4. **לא להפעיל** התערבויות אינטנסיביות בשבועיים הראשונים — נותנים זמן להסתגלות

**🟠 חריג:** ירידה > 1 SD שמתמשכת מעבר ל-3 שבועות — אסקלציה. אולי משהו קרה.

2. **פעילות הצוות התרגעות** ([cat-03](cat-03-emotion-regulation.md#מתכון-1-פינת-הרגיעה) + [cat-01](cat-01-morning-calm.md#מתכון-3-נשימת-הים)) — תשתית רגשית בלבד
3. **נוטיפיקציה למנהלת, יועצת, מתי"א, ומשרד החינוך** (פרוטוקול חירום)
4. **הסכמת הורים מורחבת** — להוסיף שאלה אישית בשאלון הורים הבא: "האם ילד.ך מדבר.ת על האירוע?"

**פעולות ב-2-4 שבועות (אחרי שיועצת מאשרת חזרה):**
5. **חזרה הדרגתית לפולס** — שאלה אחת מהממדים, לא המלאה

### Tier 1 (Universal — לכל הכיתה)

**שצריכים לרוץ בכל כיתה תמיד, לא בתגובה לבעיה:**

- בוקר רגוע — טקס פתיחה במעגל ([cat-01](cat-01-morning-calm.md))

### Tier 2 (Targeted — קבוצות קטנות / ילדים ספציפיים)

**שמופעלים על-בסיס data בפולס:**

- שייכות וקשר — כיסא חם בכיף ([cat-02](cat-02-belonging-and-connection.md))
- ויסות — פינת הרגיעה ([cat-03](cat-03-emotion-regulation.md))

- הורים — מחברת קשר ([cat-09](cat-09-parent-engagement.md))

### Tier 3 (Intensive — יחיד.ה, עם יועצת)

**רק עם אבחנה מקצועית של היועצת:**

- כל אסקלציה אישית
- שילוב מתכונים בהתאמה אישית
- שקלי ECI / תמיכה חוץ-בית-ספרית

### Meta-Tier (תשתית — לא על ילד.ה)

## 🔧 איך הדשבורד יישם זאת

הלוגיקה האוטומטית במערכת:

```python
# פסאודו-קוד של מנוע ההמלצות
def recommend_for_signal(class_pulse, child_pulse, parent_pulse, history):
    triggers = detect_triggers(class_pulse, child_pulse, parent_pulse, history)
    
    recommendations = []
    for trigger in triggers:
        # שלוף מהאינדקס 2-3 מתכונים פר טריגר
        recipes = INDEX[trigger]  # מהמיפוי שבמסמך זה
        
        # סנן לפי מה שכבר רץ בכיתה
        active = get_active_interventions(class_id)
        recipes = [r for r in recipes if r not in active]
        
        # תעדף לפי MTSS tier (התחל ב-1 לפני 2 לפני 3)
        recipes = sorted(recipes, key=lambda r: r.tier)
        
        recommendations.append({
            'trigger': trigger,
            'top_recipes': recipes[:3],
            'rationale': explain_choice(trigger, recipes[:3])
        })
    
    return recommendations
```

**עיקרון מנחה:** הדשבורד **לא** מציג 30 מתכונים. הוא מציג **2-3 ממוקדים** לסיגנל הנוכחי, עם רציונל. המורה יכולה לפתוח את הקטלוג המלא רק אם רוצה לחקור עוד.

זה ההבדל בין Aperture/DESSA (5 ההמלצות מצומצמות, נכנס למורה) לבין Panorama (900, מציף).
