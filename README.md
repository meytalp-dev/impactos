# אימפקט.OS

> מערכת ההפעלה של בית הספר. שכבת ניהול אופרטיבי + AI מעל המערכות הקיימות. לא מתחרה — משלימה.

האתר: [impact-os.app](https://impact-os.app)

---

## מבנה הריפו

```
impactos/
├── index.html                # לנדינג ראשי של המותג
├── _shared/games/            # נגני משחקים משותפים בין חבילות תוכן
├── learning-engine/          # מנוע הלמידה הדיפרנציאלי + חבילות תוכן
│   ├── index.html            # דף המוצר
│   ├── demo.html             # דמו אינטראקטיבי (חבילת תנ"ך)
│   ├── literacy/             # חבילת תוכן: אוריינות א'-ב'
│   ├── english/              # חבילת תוכן: אנגלית
│   ├── games/                # נתוני משחקים לחבילות התוכן
│   ├── annual-plans/         # תוכניות שנתיות מובנות
│   ├── research/             # מחקר תומך
│   ├── proposal-municipal.html       # הצעה למשרד החינוך
│   ├── presentation-municipal.html   # מצגת מוניציפלית
│   └── spec.html, about.html, ...
└── tanach-grade9/            # חבילת תוכן: תנ"ך ט', סוף שמואל א'
```

## פיתוח מקומי

האתר סטטי (HTML/CSS/JS). לפתיחה מקומית:

```bash
python -m http.server 8000
# או
npx serve .
```

## פריסה

האתר מתפרס אוטומטית ל-GitHub Pages בכל push ל-`main`. הדומיין `impact-os.app` מחובר דרך CNAME.

## מוצרים אחרים במשפחת אימפקט.OS

- **תנופה אנגלית** — פלטפורמת תכנית תנופה (בלייב באורט, מועבר בהמשך)
- **אסיפות הורים** — הקלטה וסיכום אוטומטי (בבדיקה)
- **אקלים** — מערכת אקלים בית ספרי (בתכנון)

## מקור

נוצר מבפנים של בית ספר. 21 מערכות חיות באורט בית הערבה. לא תיאוריה — היסטוריה תפעולית.
